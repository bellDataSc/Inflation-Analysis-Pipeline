# Inflation-Analysis-Pipeline
Pipeline de análise e previsão de inflação brasileira com modelos estatísticos ARIMA e séries temporais

Sistema de análise e previsão de índices de inflação brasileiros, com foco em IPCA. Processa dados históricos, calcula métricas econômicas e gera forecasts utilizando modelos ARIMA e técnicas de séries temporais.
